# Claim Form (Trainee Allowance)

Simple static claim form split into separate files for easier maintenance.

Files added:

- [claim-form/index.html](claim-form/index.html)
- [claim-form/css/styles.css](claim-form/css/styles.css)
- [claim-form/js/script.js](claim-form/js/script.js)

How to use:

1. Open `claim-form/index.html` in a browser.
2. Fill in trainee details and add time card entries.
3. Click "Download PDF" to save a PDF (uses html2pdf CDN).
